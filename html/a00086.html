template<typename SIMULATOR , typename POLICY , typename STATE , typename ACTION , typename SARSA_CRITIC >
std::pair<STATE,ACTION> rl::episode::adaptation ( SIMULATOR &  simulator,
const POLICY &  policy,
SARSA_CRITIC &  critic,
const STATE &  s,
const ACTION &  a 
)

This triggers an interaction from a policy. From this transition, the critic learning occurs. This function is dedicated to be used with successive transitions, since it must be given (s,a) of the last transition performed, in order to avoid a useless call to policy(s). The rl::exception::Terminal exception is raised in case of terminal transition.

Parameters
sThe current state, i.e. s = simulator.sense()
aThe action chosen by the policy, i.e. a = policy(s)
Returns
A s',a' pair, or raises an exception if a terminal transition is reached.
